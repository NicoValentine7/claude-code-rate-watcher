Claude Code Rate Watcher

Claude Code の API レート使用量をメニューバーからリアルタイム監視する macOS アプリ

Download for macOS

macOS Universal (Apple Silicon + Intel)

Features

📊

5 時間ウィンドウ追跡

5 時間のスライディングウィンドウでトークン使用量をリアルタイム表示

📅

週次制限モニタリング

168 時間の週次トークン使用量も同時に追跡

🔔

閾値アラート

使用量が 75% / 90% を超えると macOS 通知でお知らせ

⏱️

リセットカウントダウン

レート制限のリセットまでの残り時間をカウントダウン表示

Install

  1. 上のボタンから claude-code-rate-watcher-macos-universal.tar.gz をダウンロード
  2. ダブルクリックで展開し、バイナリを /usr/local/bin/ 等に配置
  3. ターミナルから claude-code-rate-watcher を実行、またはログイン項目に追加

How it Works

~/.claude/projects/ 配下の JSONL セッションファイルを監視し、 assistant レスポンスのトークン使用量を集計します。 出力トークンはコスト換算で入力の 5 倍として加重計算されます。
